## 1. 安装与启动

### 安装

1. 双击安装包（`.exe` 文件），按照安装向导完成安装
2. 安装完成后，桌面会出现「绣花厂订单管理」快捷方式

### 首次启动

- 双击桌面图标即可打开应用
- 首次使用无需密码，可在"系统设置"中设置启动密码
- 建议先到"系统设置"添加绣花机台信息，再开始录入订单

## 2. 界面概览

应用界面分为左侧导航栏和右侧主内容区：

| 工人工资 | 管理工人信息、发放工资、录入出勤 |
| 系统设置 | 机台管理、启动密码、数据备份 |

## 3. 订单管理

### 3.1 查看订单列表

进入应用后默认显示订单列表页面，提供以下功能：

- **状态筛选**：点击顶部的状态按钮（全部 / 待打样 / 打样中 / 待生产 / 生产中 / 已完成 / 已发货），快速筛选订单
- **关键字搜索**：在右上角搜索框输入订单号、产品名称或客户名称进行搜索
- **导出 Excel**：点击「导出」按钮将当前列表导出为 Excel 文件

### 3.2 新建订单

1. 点击右上角「新建订单」按钮
2. 填写订单信息：

| 字段 | 说明 | 是否必填 |
|------|------|---------|
| 订单号 | 系统自动生成（格式：XH + 年月日 + 序号） | 自动 |
| 客户 | 从下拉列表选择，也可点击 ＋ 按钮快速新建客户 | 必填 |
| 下单日期 | 默认今天，可修改 | 必填 |
| 交货日期 | 预计交货时间 | 选填 |
| 产品名称 | 如"连衣裙胸口绣花" | 必填 |
| 花版名称 | 绣花图案名称 | 选填 |
| 版号 | 花版编号 | 选填 |
| 面料类型 | 如"纯棉""雪纺" | 选填 |
| 绣花位置 | 如"前胸""后背" | 选填 |
| 绣花尺寸 | 如"10x15cm" | 选填 |
| 颜色数 | 花版使用的颜色数量 | 选填 |
| 针数 | 花版总针数 | 选填 |
| 数量 | 订单总件数 | 必填 |
| 单价（元） | 每件加工费 | 必填 |
| 定金（元） | 客户预付定金 | 选填 |
| 状态 | 默认"待打样"，可选择其他状态 | 自动 |
| 备注 | 特殊要求、注意事项 | 选填 |

3. 系统自动计算**总金额**（= 数量 × 单价）和**未收金额**（= 总金额 − 定金）
4. 点击「创建订单」保存

### 3.3 订单详情

点击订单列表中的订单号，进入订单详情页，包含三个标签页：

- **基本信息**：查看订单和绣花参数的完整信息
- **生产记录**：查看该订单关联的所有生产记录（已产数量 / 总量 / 次品数）
- **收款记录**：查看和管理该订单的收款记录，可直接「登记收款」

### 3.4 修改订单状态

在订单详情页右上角的下拉框中选择新状态即可立即更新。

订单状态流转建议：**待打样 → 打样中 → 待生产 → 生产中 → 已完成 → 已发货**

### 3.5 编辑和删除订单

- 点击「编辑」按钮修改订单信息
- 点击删除按钮（红色垃圾桶图标），确认后删除订单及其关联的生产记录和收款记录

> ⚠️ 删除操作不可撤销，请谨慎操作。

## 4. 客户管理

### 4.1 客户列表

- 显示所有客户的名称、联系人、电话、结算方式和地址
- 支持按客户名称或电话搜索

### 4.2 新建客户

1. 点击「新建客户」按钮
2. 填写客户信息：

| 字段 | 说明 |
|------|------|
| 客户名称 | 如"xx 服装厂"（必填） |
| 联系人 | 具体联系人姓名 |
| 电话 | 联系电话 |
| 地址 | 客户地址 |
| 结算方式 | 现结 / 月结 / 批结 |
| 开票信息 | 发票抬头等信息 |
| 备注 | 其他说明 |

3. 点击「保存」

### 4.3 编辑和删除客户

- 点击列表操作列的**编辑图标**修改客户信息
- 点击**删除图标**删除客户（如果该客户有关联订单，删除会失败）

### 4.4 客户详情

点击客户名称进入详情页，可查看该客户的所有关联订单。

## 5. 生产管理

### 5.1 查看生产记录

- 通过日期选择器切换日期，查看当天所有机台的生产记录
- 页面显示当日总产量和次品数汇总

### 5.2 录入产量

1. 点击「录入产量」按钮
2. 选择**机台**（从系统设置中添加的机台列表中选择）
3. 选择**订单**（仅显示"生产中"状态的订单）
4. 填写完成数量（必填）、针数、次品数
5. 点击「保存」

> 提示：如果下拉列表中没有订单，请先将对应订单的状态改为"生产中"。

### 5.3 删除记录

点击记录右侧的删除按钮，确认后删除该条生产记录。

## 6. 收款对账

### 6.1 收款记录列表

- 默认显示当月的收款记录
- 可通过**日期区间**和**搜索框**（订单号/产品/客户）筛选
- 页面底部显示筛选结果的笔数和合计金额

### 6.2 登记收款

可以从两个入口登记收款：

**入口一：收款对账页面**
1. 点击「登记收款」
2. 搜索并选择订单
3. 填写金额、日期、收款方式（现金/转账/其他）
4. 点击「保存」

**入口二：订单详情页 → 收款记录标签**
1. 点击「登记收款」
2. 填写金额、日期、收款方式
3. 点击「保存」

> 登记收款后，订单的"未收金额"会自动更新。

### 6.3 生成对账单

1. 在收款对账页面点击「生成对账单」
2. 选择客户和日期区间
3. 系统自动汇总该客户在指定期间内的订单和收款情况
4. 可导出为 Excel 文件

### 6.4 导出

点击「导出」按钮，将当前筛选结果导出为 Excel 文件。

## 7. 工人工资

工人工资模块分为三个标签页：

### 7.1 工人列表

**添加工人：**
1. 点击「添加工人」
2. 填写姓名（必填）、类型（长工/临时工）
3. 长工需填写**月薪**（元）
4. 临时工需填写**白班单价**和**夜班单价**（元/天）
5. 可选填入职日期、电话、备注
6. 点击「保存」

**编辑/删除：** 同客户管理操作方式。

### 7.2 长工工资（月薪管理）

1. 选择月份
2. 点击「生成本月工资」—— 系统根据所有在职长工的月薪自动创建工资记录
3. 发放工资后，点击 ✓ 图标**标记已发放**
4. 如需撤销，点击 ✕ 图标**标记未发放**
5. 点击「导出」导出当月工资明细

### 7.3 临时工汇总

1. 选择月份，查看该月每位临时工的出勤天数和应付金额
2. 支付完成后，点击 ✓ 图标**全部结清**该工人本月记录
3. 如需撤销，点击 ✕ 图标**取消结清**
4. 点击「导出明细」导出当月临时工工资明细

### 7.4 出勤录入

1. 在工人工资页面点击「出勤录入」按钮进入出勤页面
2. 选择日期
3. 选择临时工、班次（白班/夜班）
4. 金额会根据工人设置的日/夜班单价自动填入，也可手动修改
5. 点击「保存」

## 8. 系统设置

### 8.1 机台管理

管理绣花机台信息，录入生产记录时需要选择机台。

**添加机台：**
1. 点击「添加机台」
2. 填写名称（如"1号机"）、型号（如"田岛TMCE"）、机头数
3. 状态默认"正常"，也可设为"维修"或"停机"
4. 点击「保存」

> 机台状态为"正常"时才会出现在生产录入的选择列表中。

### 8.2 启动密码

- **设置密码**：输入新密码和确认密码，点击「设置密码」
- **清除密码**：密码栏留空，点击「清除密码」
- 设置密码后，每次启动应用需要输入密码才能进入

### 8.3 数据备份与恢复

- **一键备份**：点击后选择保存位置，导出当前数据库为 `.db` 文件
- **从备份恢复**：选择之前备份的 `.db` 文件，恢复后应用会自动重新加载

> ⚠️ 恢复操作会覆盖当前所有数据，请务必确认后再执行。

## 9. 数据安全

### 备份建议

- **建议每周至少备份一次**
- 将备份文件保存到 U 盘或其他存储设备，避免电脑硬盘故障导致数据丢失
- 超过 7 天未备份时，应用启动会弹出提醒

### 数据存储

- 所有数据保存在本机，不上传到任何服务器
- 数据库文件位于系统 AppData 目录下
- 卸载应用不会自动删除数据库文件

## 10. 常见问题

**Q：打开应用后界面空白或报错？**
A：请确认系统为 Windows 10 (1803) 或更高版本。应用依赖系统自带的 WebView2 组件，如缺失请前往微软官网下载安装。

**Q：录入生产记录时没有订单可选？**
A：只有状态为"生产中"的订单才会出现在生产录入的下拉列表中。请先到订单详情页将订单状态改为"生产中"。

**Q：删除客户时提示失败？**
A：如果该客户存在关联订单，需要先删除或修改相关订单后才能删除客户。

**Q：删除机台时提示失败？**
A：如果该机台存在关联的生产记录，需要先删除相关记录后才能删除机台。

**Q：忘记启动密码了怎么办？**
A：可以找到数据库文件 `xiuhua.db`（位于系统 AppData 目录下的应用数据文件夹），使用 SQLite 工具打开并清除 `appConfig` 表中 `key` 为 `password` 的记录。

**Q：如何迁移数据到另一台电脑？**
A：在旧电脑上使用"一键备份"导出 `.db` 文件，在新电脑安装应用后使用"从备份恢复"导入即可。

**Q：导出的 Excel 在哪里？**
A：点击导出按钮后会弹出文件保存对话框，自行选择保存位置。如未弹出对话框，请检查是否有其他窗口遮挡。
